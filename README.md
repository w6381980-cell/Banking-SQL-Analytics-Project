# Banking SQL Analytics Project

## Project Overview
This project focuses on analyzing real-time banking data using SQL.
It simulates core banking operations such as transactions, customers,
accounts, and balances to generate business insights.

## Tools & Technologies
- SQL Server
- Stored Procedures
- Views
- Joins, CTEs, Window Functions

## Key Features
- Customer-wise transaction analysis
- Daily & monthly transaction trends
- Account balance monitoring
- High-value transaction detection
- Real-time reporting using optimized SQL queries

## Database Objects Used
- Tables (Customers, Accounts, Transactions)
- Stored Procedures
- Views
- Indexes for performance optimization

## Use Cases
- Banking performance monitoring
- Fraud detection support
- Management reporting
- Real-time financial analysis

## How to Run
1. Open SQL Server Management Studio
2. Create a new database
3. Execute the SQL file:  
   `Enterprise Real-Time Analytics Dashboard.sql`
